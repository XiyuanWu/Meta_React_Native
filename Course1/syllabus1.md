# Course 1 syllabus

## Module 1  
In this module, you will get an introduction to the course and cover some roles you may pursue as a web or mobile developer. You will also learn tips on how to take this course successfully and explore a demo of the final project. As an introduction, you will learn the basics of how the internet and web pages work. Similarly, you will gain some knowledge of web servers and web browsers. Next, you will explore the evolution of mobile applications and focus on the iOS and Android platforms and programming languages. You will then concentrate on JavaScript and learn why it is named the language of the web. Finally, you will examine the differences between web and mobile browsers and find out why it is important to create responsive websites.  

## Module 2
In module 2, you will start with Hypertext Markup Language. You will learn how to create an HTML document, use simple tags and link documents to create a website. You will also cover web accessibility and learn what the Document Object Model, or DOM, is and how it is used with JavaScript to query and update web pages. Next, you will explore how to use Cascading Style Sheets, or CSS, to style web pages. You will learn how to select and style elements, use text, color, and alignment to create visually appealing web pages that offer a great user experience.  

Finally, you will explore tools that developers use within Google Chrome to make their work more efficient. You will also cover what an Integrated Development Environment, or IDE, is, identify the benefits it brings to a developer and learn about some of the most common development tools.
